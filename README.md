# 生活介護事業所 希望の家 公式ホームページ

社会福祉法人SHIPが運営する「生活介護 希望の家」の公式ホームページです。
GitHub Pagesで静的にホスティングされることを前提に制作されています。

## プロジェクト概要
- **サイト名**: 生活介護 希望の家
- **運営**: 社会福祉法人SHIP
- **目的**: 施設の紹介および見学・利用相談への誘導
- **技術構成**: HTML5, CSS3, JavaScript (Vanilla) - フレームワーク不使用

## ディレクトリ構成
- `/` - HTMLファイル群（トップページ、その他各ページ）
- `/css/` - スタイルシート（全ページ共通）
- `/js/` - 共通JavaScriptファイル（ハンバーガーメニュー等）
- `/images/` - 画像リソース

## 開発・表示確認方法
ビルド環境などは不要です。リポジトリをクローンまたはダウンロードし、`index.html` をお使いのWebブラウザで直接開くことでローカルでの表示確認が可能です。
